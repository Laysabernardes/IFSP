AULA 8 - PACKAGE

PACKAGE significa Pacotes e são uma forma de agrupamento de classes

  • Declaração de um package: 
    
package nomeDoPacote;
public class classeNormal {
    // conteudos da classe
}
  • Estrutura dos Diretórios: Se a organização das pastas estiver como por exemplo: app/ex/tp1 fica assim 
    
package app.ex.tp1;
public class classeNormal {
    // conteudo da classe
}
  • Importar: Para importar basta utilizar a palavra reservada import 
    
import com.exemplo.outropacote.OutraClasse;
public class MinhaClasse {
    // pode usar OutraClasse aqui
}
  • Para uma importação global pode usar o * 
    import com.exemplo.outropacote.*;
